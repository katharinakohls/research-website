Theses and Research Internships

You want to work on a thesis or internship on (mobile) network security and can't find something interesting here? Contact me and we'll try to find a nice project together. What is needed for this? Know your interests, what you like to do, and what you really don't want to do.

Offered

  • Implementation of a Signal Strength Measurement App, Bachelor Thesis, Internship
    Implement an Android app that allows to measure the signal strength of surrounding mobile network base stations. The thesis includes the design and implementation of the app as well as real-world measurements of signal strengths in an example area.
  • Implementation of a Downgrade Attack, Master Thesis, Internship
    Implement a downgrade attack that forces a phone from an LTE into a UMTS connection. The thesis includes C programming and maintaining a mobile network experimental setup.
  • Routing Stability, Bachelor/Master Thesis
    Conduct traceroute measurements and analyze the stability of routes in different scenarios. The thesis includes the implementation of a measurement setup a detailed analysis of the results.
  • Simulation Setup for Realistic Network Propagation, Master Thesis
    Design and implement a simulation model that can resemble real-world transmission characteristics of the Internet. This includes for example dynamic routing, multi-hop routes, or latency effects.
  • Design and Implementation of a Mobile Network Measurement Setup, Bachelor/Master Thesis
    Design and implement an experimental setup using a USRP, mobile phone, and LTE software stack to record and analyze mobile network traffic. The thesis includes creating the experimental setup as, managing a set of measurements, and analyzing the resulting traces.
